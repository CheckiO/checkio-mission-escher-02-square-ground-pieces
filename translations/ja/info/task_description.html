島へは長い旅だったのであなたは船長室で既に見つけた宝を整理することにしました。一方で遠征の数週間前に数人の大富豪と交渉しハイパーキューブが大体いくらで売れるか知っていました。

絵に描いたような土地を買うことはあなたが長い間抱いてきた希望でした。 そこに家を建て希少種の生き物を飼うことを夢見ていました。 それらには大量のお金が必要ですが近い将来実現しそうです。

'0'と'#'からなる複数行の文字列が入力データです。 '0'は空地、'#'は家の一部をあらわします。 あなたの仕事はその家を建てるのに最低限必要な長方形の土地の面積を計算することです。

入力: 家の図面

出力: 長方形の土地の総面積

Example:

{% if interpreter.slug == "js-node" %} 
house(`
0000000
##00##0
######0
##00##0
#0000#0
`) == 24
{% else %} 
house('''
0000000
##00##0
######0
##00##0
#0000#0
''') == 24
{% endif %}

どのように使われるか: 合理的な資源の使い方を計測できます

Precondition:
2x2 <= multiline string <= 10x10
