Одним из ваших давних желаний было приобретение земли в живописном месте, где вы могли бы построить дом и разводить животных редких видов. Подобное мероприятие требует весьма немалых денег, которые, скорее всего, появятся у вас в ближайшее время.

На входе ваша функция получит многострочную строку, состоящую из '0' и '#', где '0' обозначают пустые участки земли, а '#' - часть вашего дома. Ваша задача - определить площадь прямоугольного участка земли, который вам необходим, чтобы его хватило для постройки дома.

Входные данные: План дома.

Выходные данные: Размер прямоугольного участка для строительства.

Пример:

house(`

##00##0
######0
##00##0

house('''
0000000
##00##0
######0
##00##0
#0000#0
''') == 24
{% endif %}

Как это используется: Для рационального использования ресурсов.

Предусловия:
2x2 <= размер строки <= 10x10
