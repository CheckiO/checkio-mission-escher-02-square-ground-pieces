For starters, you can remove the extra elements (0) and leave only the building elements, dividing them string by string. Such methods for working with strings, like replace() and split() will help you with that.

Code will look like this: 
plan = plan.replace('0', '')
plan = plan.split('\n')
The first string replaces all '0' with an empty string (''), the second - separates the multiline string into separate strings, using the linefeed character '\n' as the separating element.
How can I calculate the area needed to build a house?
To determine the area of the required rectangular land plot you need to multiply the longest element of the plan list (the length of the house) by the number of such elements in the list (the width of the house). To determine the longest element use the max() function, and to determine the length of the list use the len() function.
